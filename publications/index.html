Invited panels, lectures & seminars

I create my conference slides in HTML5 using the awesome presentation framework reveal.js. I host them on Github and only require an up-to-date browser like Firefox or Chrome for my presentation. It’s great. I can include interactive content, I don’t need to worry about losing my usb-stick, and I can show my slides on any operating system.
